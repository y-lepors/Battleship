===== Lire attentivement ce fichier avant de jouer =====

Ce jeu est une bataille navale.
Une fois le jeu lancé il sera nécessaire de repositionner les fenêtres de grille à chaque tour de jouer, 5 secondes y seront dédiées.
À chaque tour si le joueur est humain il sera nécessaire d'entrer les coordonnées verticales puis les coordonnées horizontales.

== Placement des fichier de configuration ==
Le répertoire data où il est possible de configurer les flottes ainsi que le placement des bateaux doit être placé dans le même répertoire depuis lequel est exécuté le fichier JAR afin que le jeu fonctionne.

** Commande d'éxécution du projet : 
- Se placer dans le répertoire ou se trouve le fichier JAR
- Exécuter la commande : java battaileNavale.jar

=== Fonctionnement de l'IA ===
L'IA fonctionne d'une manière très simple, elle tire aléatoirement et est donc peu efficace. Ainsi remporter une partie contre l'IA devrait être relativement facile.
Afin de créer son fichier de placement nommé placementIA.txt l'idéal serait de demander à un ami de le remplir puisqu'il est nécessaire de faire le placement soi-même.
