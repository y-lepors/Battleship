===== Lire attentivement ce fichier avant de jouer =====

Ce jeu est une bataille navale.
Une fois le jeux lancer il sera néccessaire de repositionner les fenêtre de grille à chaque tours de de joueur, 5 secondes y seront dédiées.
A chaque tour si le joueur est humain il sera néccéssaire d'entrer les coordonnées Verticale puis les coordonnées Horizontale.

== Placement des fichier de configuration ==
Le répertoire data où il est possible de configurer les flottes ainsi que le placement des bateaux doit être placé dans le même répertoire depuis le quel est
exécuté le fichier JAR afin que le jeu fonctionne.

** Commande d'éxécution du projet : 
- Se placer dans le répertoire ou se trouve le fichier JAR
- Exécuter la commande : java battaileNavale.jar

=== Fonctionnement de l'IA ===
L'IA fonctionne d'une manière très simple, elle tire aléatoirement et est donc peut efficace. Ainsi remporter une partie contre l'IA devrait être relativement facile.
Afin de créer son fichier de placement nommé placementIA.txt l'idéal serait de demander à un amis de le remplir puisqu'il est néccessaire de faire le placement soit même.
